Palette Hex | Color Picker

Palette Hex lets you define custom web colors.

Use the following color picker to pick a color that you can use within your CSS code. The color picker provides the color values in hexadecimal and RGB. Use this color picker to find a suitable color. The slider on the right to change the hue. Click anywhere in the large square to pick a color. Once you're happy with the color, copy the hexadecimal value from the top box.
